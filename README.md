# Multi-Format-File-Conversion-and-WhatsApp-Delivery-Automation
This workflow automates file format conversion using API-based processing and delivers the converted files directly to users via WhatsApp. When a user submits a file through a form, the system detects the file type and routes it through different conversion pathways such as XLSX to PDF, DOCX to PDF, PDF to DOCX, and PDF to TXT.
